This is a prototype of a system that has already been implemented; with superior features and fullstack functionality.

## PROBLEMS
- Diffuculty in finding attachment opportunities.
- Paperwork required makes go green initiative hard.
- Time wastage.
- High institutional supervision fees.

## PROPOSED SOLUTIONS

2. A system that gives room for automated paperwork submission.
